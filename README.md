# arfetch
A simple system fetch for Arch Linux, written in bash.
## Installation
### Arch Linux
```
git clone https://github.com/AlphaNecron/arfetch.git
cd arfetch
makepkg -sri
```
### Other distros
```
git clone https://github.com/AlphaNecron/arfetch.git
cd arfetch
./install.sh
```
